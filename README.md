# 🎓 Student Result Calculator

A web-based **Student Result Calculator** that helps calculate student grades, percentages, and results based on marks entered for multiple subjects.

## 🚀 Features

- 📝 Enter marks for multiple subjects easily
- 📊 Automatic percentage and total marks calculation
- 🏆 Grade assignment based on percentage (A, B, C, D, F)
- ✅ Pass/Fail result determination
- 🖨️ Print-friendly result card
- 💡 Clean and user-friendly interface
- 📱 Responsive design — works on all screen sizes

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Page structure |
| CSS3 | Styling and layout |
| JavaScript | Logic and calculations |
| Bootstrap | Responsive UI components |

## 📐 Grading System

| Percentage | Grade | Result |
|---|---|---|
| 90% and above | A+ | Pass |
| 80% - 89% | A | Pass |
| 70% - 79% | B | Pass |
| 60% - 69% | C | Pass |
| 50% - 59% | D | Pass |
| Below 50% | F | Fail |

## ⚙️ How to Use

1. **Clone or download** the repository
2. Open `student_result_calculator.html` in any web browser
3. Enter the student's name and marks for each subject
4. Click **Calculate** to view the result
5. View the total marks, percentage, grade, and pass/fail status
6. Use the **Print** button to print the result card
